# Dynasty PropertyOS Land + Universal Property Development Engine

Purpose: expand Dynasty PropertyOS from a single house digital twin into a universal development platform for raw land, residential property, commercial property, mixed-use projects, and future subdivision/build-to-rent workflows.

Core principle: every asset starts as a Parcel, then becomes one or more Development Projects, which may contain Site Improvements, Utility Systems, Structures, Material Packages, Cost Models, Entitlements, and Digital Twin assets.

## Supported Asset Types

- Raw land
- Infill lots
- Residential single family
- Duplex / triplex / quadplex
- Multifamily
- Commercial retail
- Office
- Industrial / warehouse
- Mixed-use
- Agricultural / rural acreage
- Mobile home / manufactured housing sites
- Subdivision / land development

## Development Pipeline

1. Parcel Intake
2. Zoning + Use Feasibility
3. Site Constraints Review
4. Development Strategy Selection
5. Concept Layout
6. Sitework + Utilities Estimate
7. Vertical Construction Estimate
8. Investor / Lender Feasibility
9. Blender Site Digital Twin
10. Contractor Scope Package
11. Accounting + Job Costing Setup
12. Optional Property Passport / Blockchain Hashing
